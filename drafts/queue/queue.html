キュー構造 (Queue)

よみかた：きゅー

意味

先に入れたものが、先に出ていくという仕組みのデータ構造のこと。
専門用語で「FIFO（First In, First Out）」と呼ばれます。
レジの行列やバスの順番待ちなど、現実世界の「順番通り」の場面とそっくりです。

ポイント

  • enqueue (エンキュー) : 列の後ろに並ぶようにデータを追加します。
  • dequeue (デキュー) : 列の先頭から出るようにデータを取り出します。
  • 印刷ジョブの管理やタスク処理など、順番を守って処理したいときに最適です。

実際に体験してみよう

下のボタンを押して、キューの「先入れ先出し」を体験してみましょう。「Enqueue」で列の末尾に人を追加し、「Dequeue」で列の先頭から人が退出します。

関連用語・比較

用語 構造 特徴
キュー 先入れ先出し (FIFO) 最初に入れたものが最初に出る（レジの列）
スタック 後入れ先出し (LIFO) 最後に入れたものが最初に出る（お皿の山）
優先度付きキュー 優先度順に処理 重要なものを優先的に処理できる

コードで見てみよう (JavaScript)

JavaScriptの配列を使えば、簡単にキューを実装できます。pushで追加し、shiftで取り出すことで「先入れ先出し」を実現します。

サンプルコードを見る (JavaScript) 
// 配列を使った簡単なキュー
const queue = [];

// enqueue（追加：後ろに並ぶ）
queue.push("タスク1");
queue.push("タスク2");
queue.push("タスク3");

// dequeue（取り出し：先頭から出る）
const firstTask = queue.shift();

console.log(firstTask); // => "タスク1"
console.log(queue);     // => ["タスク2", "タスク3"]
キューとは「並んだ順に処理する」データ構造のこと。
現実の行列と同じように、順番を守って一つずつ処理するときに活躍します！
