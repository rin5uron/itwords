API

よみかた：エーピーアイ

意味

アプリケーション同士が情報をやりとりするための仕組み。
Application Programming Interface（アプリケーション・プログラミング・インターフェース）の略で、異なるシステムやサービス間でデータを交換するための決まりごと（ルール）のこと。

ポイント

  • 異なるシステム間でデータを安全に交換できる
  • 決まった形式（ルール）に従って通信するため、安定した連携が可能
  • Webサービスやスマホアプリ開発で頻繁に使用される
  • 天気予報、地図、SNSなど多くのサービスがAPIを提供している

体験してみよう

実際のAPIを呼び出して、リアルタイムの情報を取得してみましょう。
※無料のAPIサービスを使用しています。

ランダムな名言を取得します

JavaScriptで実践してみよう

JavaScriptのfetch関数を使うと、簡単にAPIを呼び出すことができます。以下のサンプルコードで、その実装方法を確認してみましょう。

サンプルコードを見る (JavaScript) 
async function startApiDemo() {
  const messageDiv = document.getElementById('demo-message');
  messageDiv.innerHTML = '
APIを呼び出し中...
';
  
  try {
    // APIエンドポイントにリクエストを送信
    const response = await fetch('https://api.quotable.io/random?minLength=50');
    
    // レスポンスをJSONとして解析
    const data = await response.json();
    
    // 取得した名言を表示
    messageDiv.innerHTML = `
      

        
"${data.content}"

        
- ${data.author}

      

    `;
  } catch (error) {
    messageDiv.innerHTML = '
エラーが発生しました。
';
  }
}
