# 今日の作業を開始

`docs/WORKFLOW.md` の「毎日の作業ルーチン」を読み込んで、30分チャレンジを開始してください。

## 実行手順

1. `docs/WORKFLOW.md` の「毎日の作業ルーチン」セクションを読み込む
2. Phase 1（データ分析5分）から順番に実行
3. 各Phaseで完了後、次のPhaseに進む
4. 全て完了したら「今日の作業完了」と報告

## 重要

- ユーザーに「何をしますか?」と聞かない
- ワークフローの指示に従って自動的に実行
- 各Phaseの完了条件を満たしてから次に進む
- 作業履歴は GitHub Issue にコメントとして記録
