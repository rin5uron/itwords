# コトネちゃん｜StudioJinsei 代表キャラクター設定

**作成日：2025/12/20**

StudioJinseiの代表キャラクター「コトネちゃん」のビジュアル設定です。
ロゴ、サイト、LINE、SNS等、すべてのビジュアルで共通して使用します。

## 🎭 基本設定

### 役割・立ち位置
| 項目 | 設定 |
|------|------|
| **名前** | コトネちゃん |
| **役割** | StudioJinsei 代表 |
| **立ち位置** | プロフェッショナル、でも親しみやすい |

| **年齢層** | 20代後半〜30代前半の落ち着き |

## 👤 ビジュアル設定

### 外見
| 項目 | 設定 |
|------|------|
| **髪** | ダークロング、ナチュラル |
| **服装** | ホワイト系、エレガント |
| **アイコン** | ミントグリーンのバケットハット（葉柄） |
| **スタイル** | 手書き風キレイめ |

### 線・質感
| 項目 | 設定 |
|------|------|
| **線の質感** | 柔らかい、少しラフ、温かみ |
| **色味** | 落ち着いた、洗練、トーン控えめ |
| **背景** | ミニマル、余白たっぷり |

### 表情の原則
- 穏やかな微笑み（強すぎない）
- 信頼感を与える（プロとして）
- 親しみやすさ（距離を縮める）
- 「次、動けそう」と思わせる安心感

## 📐 使用場面別ガイド

### ロゴと一緒に使う場合
- 上半身またはアイコンのみ
- シンプルなシルエット
- ミントグリーンのバケットハットを強調
- 小サイズでも認識できるデザイン

### サイトヘッダー・ヒーローセクション
- 全身または上半身
- 余白を活かしたレイアウト
- StudioJinseiのコンセプトを体現するポーズ
- 柔らかいグラデーション背景

### LINE・SNS用
- 正方形または縦長に対応
- 顔がはっきり見えるサイズ
- 表情豊かに
- 背景は白またはペールミント

### 名刺・印刷物
- シンプルなアイコン形式
- モノクロでも認識できるデザイン
- 細部は簡略化

## 🚫 避けるべき表現

### ビジュアル面
- 過度に装飾的（キラキラ、エフェクト過多）
- アニメ・漫画すぎる表現（デフォルメ強すぎ）
- リアルすぎる写実表現
- 安っぽいクリップアート風

### 表情・ポーズ
- 過度に元気すぎる（テンション高すぎ）
- 無表情・冷たい印象
- セクシャルな表現
- 幼すぎる・子供っぽすぎる

---

## 🔄 バリエーション展開

### 季節・イベント対応
- 基本デザインは変えない
- 小物（帽子、アクセサリー）で季節感
- カラーパレットは維持

### シチュエーション別
- オフィスシーン：デスク、PC
- 相談シーン：椅子に座って傾聴
- 案内シーン：手を差し伸べる、指し示す
