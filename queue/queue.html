キュー構造 (Queue)

よみかた：きゅー

意味

先に入れたものが、先に出ていくという仕組みのデータ構造。
先入れ先出し、「FIFO（First In, First Out）」とも呼ばれる。

ポイント

  • enqueue (エンキュー) : 列の後ろに並ぶようにデータを追加する。
  • dequeue (デキュー) : 列の先頭から出るようにデータを取り出す。
  • 印刷ジョブの管理やタスク処理など、順番を守って処理したいときに最適。

体験してみよう

下のボタンを押して、キューの「先入れ先出し」を体験してみましょう。「Enqueue」で列の末尾に人を追加し、「Dequeue」で列の先頭から人が退出します。

JavaScript配列で実践してみよう

JavaScriptで簡単にキューを実装できます。pushで追加し、shiftで取り出すことで「先入れ先出し」を実現します。

サンプルコードを見る (JavaScript) 
// 配列を使った簡単なキュー
const queue = [];

// enqueue（追加：後ろに並ぶ）
queue.push("タスク1");
queue.push("タスク2");
queue.push("タスク3");

// dequeue（取り出し：先頭から出る）
const firstTask = queue.shift();

console.log(firstTask); // => "タスク1"
console.log(queue);     // => ["タスク2", "タスク3"]

      <section class="term-intro">
        <h2>意味</h2>
        <p>
          <strong>先に入れたものが、先に出ていく</strong>という仕組みのデータ構造のこと。<br>
          専門用語で「<strong>FIFO（First In, First Out）</strong>」と呼ばれます。<br>
          レジの行列やバスの順番待ちなど、現実世界の「順番通り」の場面とそっくりです。
        </p>
      </section>

      <section class="term-point">
        <h2>ポイント</h2>
        <ul>
          <li><code>enqueue</code> (エンキュー) : <strong>列の後ろに並ぶ</strong>ようにデータを追加します。</li>
          <li><code>dequeue</code> (デキュー) : <strong>列の先頭から出る</strong>ようにデータを取り出します。</li>
          <li>印刷ジョブの管理やタスク処理など、<strong>順番を守って処理したいとき</strong>に最適です。</li>
        </ul>
      </section>

        <h2>意味</h2>
        <p>
          <strong>先に入れたものが、先に出ていく</strong>という仕組みのデータ構造のこと。<br>
          専門用語で「<strong>FIFO（First In, First Out）</strong>」と呼ばれます。<br>
          レジの行列やバスの順番待ちなど、現実世界の「順番通り」の場面とそっくりです。
        </p>
      </section>

      <section class="term-point">
        <h2>ポイント</h2>
        <ul>
          <li><code>enqueue</code> (エンキュー) : <strong>列の後ろに並ぶ</strong>ようにデータを追加します。</li>
          <li><code>dequeue</code> (デキュー) : <strong>列の先頭から出る</strong>ようにデータを取り出します。</li>
          <li>印刷ジョブの管理やタスク処理など、<strong>順番を守って処理したいとき</strong>に最適です。</li>
        </ul>
      </section>

  <script>
    let queue = [];
    let personCount = 0;
    const visualizer = document.getElementById('queue-visualizer');
    const message = document.getElementById('demo-message');

    function enqueueItem() {
      if (queue.length >= 5) {
        message.textContent = "行列が長すぎます！";
        message.style.color = "#d9534f";
        return;
      }
      personCount++;
      const newPerson = `客 ${personCount}`;
      queue.push(newPerson);
      renderQueue();
      message.textContent = `"${newPerson}" が列に並びました。`;
      message.style.color = "#333";
    }

    function dequeueItem() {
      if (queue.length === 0) {
        message.textContent = "誰も並んでいません。";
        message.style.color = "#d9534f";
        return;
      }
      const servedPerson = queue.shift();
      renderQueue();
      message.textContent = `"${servedPerson}" がレジを終えました。`;
      message.style.color = "#333";
    }

    function renderQueue() {
      visualizer.innerHTML = '';
      queue.forEach(item => {
        const div = document.createElement('div');
        div.className = 'queue-item';
        div.textContent = item;
        visualizer.appendChild(div);
      });
    }
    
    // 初期状態
    enqueueItem();
    enqueueItem();
  </script>
