## サイト実装5分の学習内容

### 2026-01-24
**実施内容**: SMTP、IPアドレスとポート番号の用語ページ作成

**学んだこと**:
- テンプレート（`docs/templates/page-template.tsx`）に完全準拠することの重要性
- 用語ページ作成時の必須チェックリスト：
  1. README.mdを読む
  2. テンプレートファイルを読む
  3. 既存ページを参考にする
  4. `TableOfContents`と`PageSummary`を必ず追加
  5. セクション順序を守る（概要 → 日常生活での実例 → 補足 → 関連用語 → FAQ）
  6. トップページ（`app/page.tsx`）にカードを追加
  7. 検索インデックスを更新（`npm run build-index`）

**実装したこと**:
- SMTPページ作成（`app/terms/smtp/page.tsx`）
- IPアドレスページにポート番号の説明を追加
- HTTPページに関連用語リンクを追加
- トップページに新規ページのカードを追加
- 検索インデックスを更新

**反省点**:
- 最初にテンプレートを確認せずに作業を開始してしまった
- トップページへの追加を忘れていた
- `.claude/TEMPLATE_RULES.md`を作成して、今後は自動的にテンプレートに沿うようにルール化
