# Issue #38 アフィリエイト配置メモ

**最終更新**: 2026-02-09

## 表示されるページと場所

| 種類 | 表示されるURL | 場所・条件 |
|------|----------------|------------|
| **基本情報の本（楽天＋Amazon）** | **トップページ `/`** | 「人気の用語」セクションの**右下のカード**（PRタグ・表紙・楽天で見る \| Amazonで見る） |
| **楽天バナー（A8）** | **用語ページ**（例: `/terms/api`, `/terms/cve`, `/terms` 一覧も含む） | **PCのみ**（幅1024px以上）、記事の**右横**サイドバー |
| **関連おすすめ書籍（Amazon）** | **用語ページ**（上記と同じ） | **FAQの下**・フッター広告の上（白いコンテナ内） |

## 確認手順（ローカル）

1. `npm run dev` で起動し、`http://localhost:3000` を開く。
2. **トップ**: 下までスクロールし「人気の用語」の**6枚目（右下）**に基本情報の本のPRカードがあるか確認。
3. **用語ページ**: `http://localhost:3000/terms/api` を開く。
   - **PC幅（1024px以上）**: 画面右に楽天バナー（縦長）が出る。
   - 一番下までスクロール: 「関連おすすめ書籍」→ フッター上広告の順で出るか確認。
4. 本番（デプロイ先）で見る場合は、**この変更をデプロイしたあと**で同じURLを開いて確認する。

## 注意

- 楽天サイドバーは **768px～1023px では非表示**（1024px以上で表示）。
- HTTPSサイトで楽天バナー画像は `https://hbb.afl.rakuten.co.jp/...` を使用（混在コンテンツ防止）。
