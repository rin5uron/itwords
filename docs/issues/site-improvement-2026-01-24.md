## 📚 学んだこと

### テンプレート準拠の重要性
- **テンプレートファイル**: `docs/templates/page-template.tsx`を必ず確認すること
- **README**: `readme.md`の103行目にテンプレート使用の記載がある
- **既存ページ**: 同じ構造の既存ページ（例：`app/terms/http/page.tsx`）を参考にする

### 用語ページ作成時の必須チェックリスト
1. ✅ `readme.md`を読む
2. ✅ `docs/templates/page-template.tsx`を読む
3. ✅ 既存の類似ページを参考にする
4. ✅ `TableOfContents`コンポーネントをインポート・配置
5. ✅ `PageSummary`コンポーネントをインポート・配置
6. ✅ セクション順序を守る（概要 → 日常生活での実例 → 補足 → 関連用語 → FAQ）
7. ✅ 関連用語セクションに`className="term-comparison"`を付与
8. ✅ 関連用語テーブルに`className="comparison-table"`を付与
9. ✅ **トップページ（`app/page.tsx`）にカードを追加**
10. ✅ **検索インデックスを更新（`npm run build-index`）**

## ⚠️ 反省点

### 最初の作業で見落としたこと
1. **テンプレートを確認せずに作業を開始**
   - 最初に`docs/templates/page-template.tsx`を読むべきだった
   - READMEにもテンプレート使用の記載があるのに確認しなかった

2. **トップページへの追加を忘れていた**
   - 用語ページを作成したら、必ずトップページにカードを追加する必要がある
   - 検索インデックスも更新する必要がある

3. **セクションの順序が違っていた**
   - 「日常生活での実例」が後半にあった
   - テンプレートでは2番目のセクションとして配置すべき
