## 技術的な制約と方針

### 一括指定は枠外しかできない

- `layout.tsx` では `{children}` の前後しか要素を置けない
- ページ内の任意の位置（例: 関連用語とFAQの間）には layout から差し込めない
- そのため **フッター上** は layout で一括配置が可能

### ページに最適化するには記事内広告がよい

- **記事内広告**は枠のサイズに合わせて表示される
- コンテンツ幅（max-width: 800px）に合わせて表示できる
- ページの文脈に馴染みやすい
- スマホ時の高さを CSS で制御しやすい（目次の下: 6vh、フッター上: 10vh）
