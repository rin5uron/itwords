## SEO超基礎知識

### SEOとは？
**SEO（Search Engine Optimization）** = 検索エンジン最適化

**簡単に言うと**: Googleで検索したときに、自分のサイトが上位に表示されるようにする工夫。

### なぜSEOが必要？
- **上位表示 → アクセス増加 → 収益増加**
- 1位と10位では、**クリック率が約10倍違う**

### SEOの3本柱
1. **技術的SEO**: サイトの構造・速度・モバイル対応
2. **コンテンツSEO**: 記事の質・キーワード・文字数
3. **外部SEO**: 他サイトからのリンク（被リンク）

#### ページスピードインサイトは調べる必要ある？

**結論: 調べておくとよい（優先度は中）**

- **Page Speed Insights** = ページの表示速度を測るGoogleの無料ツール（https://pagespeed.web.dev/）

- **Next.jsなら**: 静的生成や最適化済みのことが多いので、大きな問題は出にくい

#### 実際のサイト（itwords.jp）のレイアウト改善案（分析結果に基づく）

以下は、Page Speed Insights の指摘を受けた**実際のページ**の改善案。

| 指摘項目 | 想定原因 | 改善案 |
|----------|----------|--------|
| **CLS 0.406（レイアウトシフト）** | 広告（AdSense）が読み込み後に高さが決まる | 広告枠に `min-height` を指定して読み込み前からスペースを確保 |
| **CLS** | Font Awesome（CDN）の読み込みでアイコンが後から表示される | `font-display: swap` の確認、またはアイコンより上に `preload` を追加 |
| **LCP 3.0秒** | ヒーロー画像の読み込み | すでに `priority` 指定あり。元画像を WebP に変換して軽量化 |
| **画像の次世代フォーマット** | JPG/PNG のまま | Next.js の `Image` は自動最適化あり。`next.config` で WebP 出力を有効化 |
| **レンダリングブロック** | Font Awesome の CSS が head で同期的に読み込まれている | `rel="preload"` で非同期読み込み、または `media="print" onload="this.media='all'"` で遅延読み込み |
| **アクセシビリティ（コントラスト）** | 淡い色のテキスト・背景の組み合わせ | 該当箇所の色のコントラスト比を 4.5:1 以上に調整 |
| **ブラウザコンソールエラー** | 広告や外部スクリプト由来の可能性 | DevTools でエラー内容を確認し、原因に応じて対処 |

**実装の優先順位（余裕があれば）**:
1. 広告枠の `min-height` 指定（CLS 対策で効果が大きい）
2. ヒーロー画像の WebP 化
3. Font Awesome の読み込み最適化
4. コントラストの確認・修正

**あとで実施**: [#35 広告枠 - CLS対策（min-height指定）](https://github.com/rin5uron/itwords/issues/35)

## SEO用語集

### 技術的SEO
- **サイトマップ**: Googleに全ページを伝えるファイル
- **robots.txt**: Googleに「ここは見ないで」を伝えるファイル
- **構造化データ**: Googleにサイト情報をわかりやすく伝える仕組み
- **インデックス**: Googleがページを検索結果に登録すること

### コンテンツSEO
- **キーワード**: ユーザーが検索するワード
- **ロングテール**: 複数ワードの組み合わせ（競合が弱い）
- **title**: ページタイトル（50文字以内推奨）
- **description**: ページ説明文（120文字以内推奨）
- **H2見出し**: 記事の構造（SEOに重要）
- **内部リンク**: 自サイト内の別ページへのリンク
- **外部リンク**: 他サイトへのリンク
- **E-E-A-T**: Experience（経験）、Expertise（専門性）、Authoritativeness（権威性）、Trustworthiness（信頼性）

### 分析
- **PV（Page View）**: ページが見られた回数
- **セッション**: サイト訪問回数
- **CTR（Click Through Rate）**: クリック率（表示回数に対するクリック数の割合）
- **掲載順位**: Google検索結果での順位
